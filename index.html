PORTFÓLIO

My Pet Friend

My Pet Friend é uma aplicação web fullstack desenvolvida no curso técnico em Informática do Senac, com foco em tornar a adoção de animais mais consciente e responsável. A plataforma conecta usuários a ONGs e protetores, permitindo o cadastro de instituições e pets, além de contar com um quiz interativo que indica o tipo de animal ideal conforme o perfil do usuário. O projeto foi desenvolvido com HTML5, CSS3, Bootstrap, JavaScript e TypeScript. Atualmente, o servidor está desligado , mas todo o código do frontend e backend está disponível no Github.

Cafeteria Lunna

Cafeteria Lunna é um site básico desenvolvido para uma cafeteria fictícia, criado com o objetivo de praticar HTML, CSS e JavaScript. O projeto conta com página inicial com banner em carrossel, apresentação da cafeteria, menu de produtos em PDF, seção de itens mais vendidos e área de contato com mapa incorporado.
